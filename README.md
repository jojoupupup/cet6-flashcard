# 🎓 六级背单词 - 便携版

基于艾宾浩斯记忆曲线的 CET-6 六级高频词背诵应用，包含 498 个六级高频词。

---

## 🚀 快速开始

### 方式一：一键启动（推荐）

1. 双击 **启动.bat**
2. 自动检测 Python 或 Node.js 并启动服务器
3. 浏览器自动打开，手机扫码即可使用

### 方式二：直接打开

1. 双击 **背单词.html** 直接在浏览器中使用
2. 功能完全可用，但无法安装为手机App

### 方式三：手动启动服务器

**有 Python 的情况下：**
```
python start_server.py
```

**有 Node.js 的情况下：**
```
node start_server_node.js
```

---

## 📱 安装为手机App（PWA）

1. 手机浏览器打开服务器地址（启动后会显示）
2. 浏览器菜单 → **「添加到主屏幕」** 或 **「安装应用」**
3. 桌面出现App图标，像原生App一样使用

---

## 🌐 公网部署（让任何人都能访问）

如果你想让没连局域网的人也能使用，可以部署到 GitHub Pages：

1. 在 GitHub 上创建一个新仓库（如 `cet6-flashcard`）
2. 把所有文件上传到仓库
3. 仓库 Settings → Pages → Source 选 `main` 分支
4. 访问链接就是 `https://你的用户名.github.io/cet6-flashcard/背单词.html`

## 🧠 功能特点

- ✅ 艾宾浩斯记忆曲线：0→1→2→4→7→15→30天间隔复习
- ✅ 每个单词：音标 + 词性释义 + 英中例句
- ✅ 学会了/还不会按钮 + 手势滑动操作
- ✅ 三次学会 = 完全掌握，进入复习周期
- ✅ 未掌握单词循环出现
- ✅ 连续学习天数（Streak）
- ✅ 数据自动保存到浏览器
- ✅ 离线可用（PWA）
- ✅ 震动反馈 + 发音朗读
